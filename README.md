# SOC Analyst Portfolio

## Objective
Hi, I'm Taiwo. This repository serves as a central portfolio showcasing hands-on Security Operations Center (SOC) projects. I developed an interest in cybersecurity and decided to focus on becoming a SOC Analyst (Tier 1). These projects show my ability to monitor logs, analyze alerts and investigate basic security incidents using tools commonly used in real-world environments.
